# Introducción

La primera unidad de la asignatura **Teoría de Programación** introduce los conceptos fundamentales que sirven como base para el desarrollo de software. Durante esta unidad se estudian los principios de la lógica computacional, la representación de algoritmos mediante pseudocódigo y diagramas de flujo, así como la importancia de validar soluciones antes de implementarlas en un lenguaje de programación.

El dominio de estos contenidos permite desarrollar el pensamiento lógico y la capacidad para resolver problemas de manera estructurada, competencias esenciales para la formación de un profesional en Computación.

# Objetivos

## Objetivo general

Comprender los fundamentos de la lógica de programación mediante el análisis, diseño y representación de algoritmos para resolver problemas computacionales sencillos.

## Objetivos específicos

- Comprender el concepto de algoritmo y su importancia.
- Elaborar pseudocódigos utilizando una estructura organizada.
- Representar soluciones mediante diagramas de flujo.
- Aplicar pruebas de escritorio para verificar algoritmos.
- Diferenciar los principales tipos de lenguajes de programación.
- Resolver problemas sencillos utilizando programación estructurada.
